## Spring security

-----

### **Описание:**

- Приложение с регистрацией, авторизацией, восстановлением пароля через почту

- Выдача ролей и статуса пользователей

- Метод секьюрити

- Кастомные валидаторы

-----

### **Технологии:**

  - База данных MySQL

  - Spring Security, Mail, Validation, Data, Boot, MVC

  - Maven, Lombok,
